\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classHArDCore3D_1_1UVector}{UVector}} class describes a vector of unknowns for discrete functions defined by polynomials in the cells and polynomials on the faces (polynomials on the edges are not taken into account yet, but this can easily be changed). The class describes the respective degrees of cell and faces polynomials considered. In the values stored in an element of this class, the coefficients on the cell basis functions come first (in the order of the cells), and then all the face basis functions (in the order of the faces)
