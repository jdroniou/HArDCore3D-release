\doxysubsection{Detailed Description}
Discrete Hdiv space\+: local operators, L2 product and global interpolator. 

On each face, the DOFs correspond to the polynomial bases on the face provided by m\+\_\+ddr\+\_\+core. On each element, the DOFs are first those of the $\mathcal{G}^{k-1}$ component and then of the $\mathcal{G}^{c,k}$ component, each one of them following the bases of these spaces provided by m\+\_\+ddr\+\_\+core.
