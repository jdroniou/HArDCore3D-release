\doxysubsection{Detailed Description}
A structure to store the serendipity, extension and reduction operators. 

The extension operators do \mbox{\hyperlink{classHArDCore3D_1_1SXGrad}{SXGrad}} -\/$>$ \mbox{\hyperlink{classHArDCore3D_1_1XGrad}{XGrad}}. The reduction operators do not produce all unknowns, only the ones in the mesh entities they\textquotesingle{}re associated to; so they do \mbox{\hyperlink{classHArDCore3D_1_1XGrad}{XGrad}} -\/$>$ $P^{\ell_P}(P)$ where P is a face or an element (the only use of the reduction operator is to compute the interpolate, and this version is what is useful).
