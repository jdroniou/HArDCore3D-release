\doxysubsection{Detailed Description}
Base class for global D\+OF spaces. Provides functions to manipulate global D\+O\+Fs (the local version being provided by D\+O\+F\+Space). 

The D\+O\+Fs are organised by increasing geometric entities dimensions\+: D\+O\+Fs of vertices, D\+O\+Fs of edges, D\+O\+Fs of faces, D\+O\+Fs of cells.
