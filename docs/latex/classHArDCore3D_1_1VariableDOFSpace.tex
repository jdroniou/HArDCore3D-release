\doxysubsection{Detailed Description}
Base class for global D\+OF spaces. 

The D\+O\+Fs are organised by increasing geometric entities dimensions\+: D\+O\+Fs of vertices, D\+O\+Fs of edges, D\+O\+Fs of faces, D\+O\+Fs of cells. This version allows for cases where the number of D\+OF on each mesh entity varies from one mesh entity to the next
