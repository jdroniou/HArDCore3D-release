The \mbox{\hyperlink{classHHO3D}{H\+H\+O3D}} class contains methods applicable to a general 3D H\+HO model. It contains assemble and solve routines usable by any scheme. The global operator and load vector are passed to the model through setters and are then accessible by assemble and solve.
