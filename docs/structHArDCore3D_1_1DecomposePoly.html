Detailed Description

template<typename BasisType>
struct HArDCore3D::DecomposePoly< BasisType >

Structure to decompose a set of polynomials on a basis on a face or a cell.

The main interest of this structure for faces is to give a decomposition of traces (scalar, normal, etc.) of cell polynomials on a basis of face polynomial. This enables the usage of these traces in a face GramMatrix. However, the decomposition leads to increased rouding errors compared to a straight usage of compute_gram_matrix. Some tests however seem to indicate that this version of DecomposePoly is relatively robust up to degrees ~4 or 5
