Detailed Description

Virtual H1 space: local operators, L2 product and global interpolator.

The order of the DOFs in each element are:

Mesh entity Type Space Description
V function \(\mathbb{R}\) vertex value of \(q\)
E function \(P^{k-1}(E)\) projection of \(q\)
F gradient \(R^{c,k+1}(F)\) projection of \(\mathrm{grad}_Fq\)
T gradient \(R^{c,k}(T)\) projection of \(\mathrm{grad}q\)

No serendipity here (hence the \(R^{c,k+1}\) on the faces)
