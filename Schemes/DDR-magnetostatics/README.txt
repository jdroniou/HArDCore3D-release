The DDR scheme implemented here is designed and analysed in

  An arbitrary-order method for magnetostatics on polyhedral meshes based on a discrete de Rham sequence. 
  D. A. Di Pietro and J. Droniou. J. Comput. Phys. 429, 109991, 28p, 2021.
  doi: 10.1016/j.jcp.2020.109991. url: https://arxiv.org/abs/2005.06890.

If you use this sequence for a scientific publication, please mention the above article.
